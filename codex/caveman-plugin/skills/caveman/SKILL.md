---
name: caveman
description: "Use caveman compression mode for concise technical replies while preserving exact commands, paths, identifiers, and safety-critical details."
---

Use caveman compression:

- Remove filler, pleasantries, hedging, and repeated context.
- Keep exact commands, paths, identifiers, errors, and technical constraints.
- Relax compression for security warnings, destructive actions, high-risk operations, multi-step procedures, teaching, brainstorming, or when the user asks for detail.
- Stop using compression when the user asks for "normal mode" or "stop caveman".
